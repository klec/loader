Для установки этого лоадера вам нужно:

  1. загрузить loader.js
  2. добавить его на свою страницу <script src="loader.js"></script>
  3. заменить стандартную картинку лоадера например <img srg="/path/to/loader.gif"> на следующий тег <svg class="loader" style="position: absolute; display: block; left: 50%; top:50%; transform: translateX(-50%) translateY(-50%);"></svg>
  4. по желанию стиль SVG можете вынести в отдельный файл.
Загрузчиков на странице может быть несколько.

To install this loader you need to:

  1. download loader.js
  2. add it to your page <script src="loader.js"></script>
  3. replace common loader image <img srg="/path/to/loader.gif"> by this tag <svg class="loader" style="position: absolute; display: block; left: 50%; top:50%; transform: translateX(-50%) translateY(-50%);"></svg>
  4. if want to, you can move style of SVG into separate file
The loaders can be added few times on a single page
